ClaudeBoard

Monitors Claude Code sessions.

Status

  • idle — waiting for input
  • working — thinking or running tools
  • stopped — session inactive for 5+ minutes

Controls

  • − — remove project from tracking
  • ? — open this help window

CLI

  • claudeboard add . — track current directory
  • claudeboard add /path — track specific path

Only shows explicitly tracked projects. Refreshes on changes.
